# Admin Access URLs

## Try These URLs:

### Main Admin Dashboard:
https://jewellery-store-hfug.vercel.app/admin/dashboard

### Admin Login:
https://jewellery-store-hfug.vercel.app/admin/login

### Alternative Routes:
https://jewellery-store-hfug.vercel.app/admin
https://jewellery-store-hfug.vercel.app/admin/dashboard

## If 404 Error:
The admin routes might not be properly configured in the build.

## Troubleshooting:
1. Try the main dashboard URL first
2. If that works, the login route might have an issue
3. Check Vercel deployment logs for routing errors
