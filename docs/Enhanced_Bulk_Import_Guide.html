📦 Enhanced Bulk Product Import Guide

Complete guide to importing products with all advanced features including category-specific sizing

🎯 Overview

The Enhanced Bulk Import system allows you to import 100+ products with all the same features as the Add Product form, including:

  • ✅ Category-specific sizing (rings, bangles, necklaces, etc.)
  • ✅ Complete product specifications (materials, occasions, weight, dimensions)
  • ✅ Cost management (purchase price, packaging, logistics)
  • ✅ Advanced product details (SKU, rating, badges)
  • ✅ Multiple images per product
  • ✅ Inventory management

📋 Complete Field Reference

🆔 Basic Information

id: Unique product ID
name: Product name
category: Jewelry category
description: Product description
sku: Product SKU code

💰 Pricing

price: Sale price
originalPrice: Original price
rating: Customer rating (1-5)

📦 Inventory

inStock: true/false
stockCount: Available quantity
isNewArrival: true/false

🏷️ Display Options

badge: NEW, SALE, etc.
badgeColor: bg-green-500, bg-red-500
isFeatured: true/false

📏 Specifications

sizes: Category-specific sizes
materials: Material types
occasions: Usage occasions
weight: Product weight
dimensions: Size dimensions

💸 Cost Management

cost_purchasePrice: What you paid
cost_packagingCost: Packaging
cost_logisticsCost: Shipping
cost_otherCharges: Other costs

🖼️ Images

images: Image URLs (semicolon separated)
Format: "url1;url2;url3"

💡 Size Usage Tips:

  • Multiple Sizes: Separate with semicolon: "6;7;8"
  • Single Size: Just one value: "Free Size"
  • No Sizes: Leave empty for default sizing
  • Category Match: Sizes must match category options

📝 Step-by-Step Import Process

Step 1: Download Template

Download the enhanced CSV template with all fields pre-configured.

📥 Download Enhanced Template

Step 2: Prepare Your Data

Fill in the CSV file with your product information. Use the field reference above to ensure correct formatting.

Step 3: Upload CSV File

Go to Admin Dashboard → Bulk Import → Choose File → Upload your prepared CSV.

Step 4: Review Preview

The system will show a preview of your products. Review for any errors or missing information.

Step 5: Import Products

Click "Import Products" to add all products to your inventory with full details.

🎯 Field Formatting Guide

📊 Number Fields

Format: Plain numbers
Examples: 2999, 25, 4.5
Fields: price, stockCount, rating, cost fields

✅ Boolean Fields

Format: true/false
Examples: true, false
Fields: inStock, isNewArrival, isFeatured

🏷️ Multiple Values

Format: Semicolon separated
Examples: "6;7;8", "Gold;Silver"
Fields: sizes, materials, occasions, images

📝 Text Fields

Format: Quoted text
Examples: "Product Name", "Description"
Fields: name, description, badge, sku

⚠️ Important Guidelines

⚠️ Critical Requirements:

  • Required Fields: name, category, price, stockCount
  • Category Validity: Category must exist in your system
  • Size Consistency: Sizes must match category options
  • Image URLs: Must be valid, accessible URLs
  • Unique IDs: Each product needs a unique ID
  • CSV Format: Use proper CSV formatting with quotes

💡 Best Practices:

  • Test Small: Start with 5-10 products to test
  • Backup Data: Backup your current inventory first
  • Validate URLs: Test image URLs before import
  • Check Categories: Ensure all categories exist
  • Review Preview: Always check the preview before importing
  • Cost Accuracy: Double-check cost calculations
