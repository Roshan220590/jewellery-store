🎛️ Managing Products via Dashboard (RECOMMENDED)

Primary Method: Use the Admin Dashboard for all product management

Adding New Products:

  1. Go to Admin Dashboard → Products
  2. Click "Add New Product" button
  3. Fill in product details:
    • Name and description
    • Category selection
    • Price and original price
    • Stock quantity
    • Images (upload or URL)
    • Sizes, materials, occasions
  4. Click "Save Product"
  5. Product appears immediately on website

💰 Profit Management

📊 Profit Tracking System

What's Tracked:

  • Purchase Price: What you paid for the item
  • Additional Costs: Packaging, logistics, other charges
  • Total Cost: All costs combined
  • Selling Price: Customer price

💾 Backup Methods

Method 1: Dashboard Export (Recommended)

  1. Go to Admin Dashboard → Settings → Data & Storage
  2. Click "Export Full Data"
  3. Choose format (JSON recommended)
  4. Save file to backup location
  5. Name with date: backup-2024-03-28.json

Method 2: Browser Storage Backup

  1. Open browser Developer Tools (F12)
  2. Go to Application → Local Storage
  3. Copy all jewellery-store data
  4. Save to text file

Method 3: File System Backup

  1. Copy entire project folder
  2. Paste to backup location
  3. Rename with date
  4. Include node_modules if needed

🛠️ Troubleshooting

🚨 Common Issues & Solutions

💥 Admin Dashboard Not Loading

  1. Check terminal for error messages
  2. Ensure npm run dev is running
  3. Clear browser cache (Ctrl+F5)
  4. Check for JavaScript errors in console
  5. Restart development server

🔐 Login Issues

  1. Verify email: rs.jewellery.shop22@gmail.com
  2. Verify password: admin123
  3. Check if localStorage has admin data
  4. Clear browser data and try again

📦 Products Not Showing

  1. Check centralDataManager.js for syntax errors
  2. Verify product IDs are unique
  3. Check category names match exactly
  4. Ensure inStock is true
  5. Check browser console for errors

💾 Data Not Saving

  1. Check if localStorage is enabled
  2. Verify browser isn't in private mode
  3. Check for storage quota limits
  4. Clear old data and try again

🖼️ Images Not Loading

  1. Test image URLs in browser
  2. Check for typos in URLs
  3. Ensure images are publicly accessible
  4. Check file paths for local images
  5. Verify image formats (JPG, PNG)

🔧 Error Messages

Most Common Errors:

  • "Unexpected token" - Check for missing commas, brackets
  • "Cannot read property" - Check for undefined objects
  • "Network error" - Check server connection
  • "Storage quota exceeded" - Clear browser data
